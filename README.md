# 🍔 Foodie's Hub

A modern and responsive food ordering website built using **HTML**, **CSS**, and **JavaScript**. This project showcases a clean user interface where users can browse food items, filter by categories, add items to the cart, and view the total price—all without using a database or backend.

## 🚀 Features

* 🍕 Browse a variety of food items
* 🔍 Filter menu items by category
* 🛒 Add and remove items from the shopping cart
* 💰 Automatic total price calculation
* 📱 Fully responsive design for mobile, tablet, and desktop
* ⚡ Fast and lightweight (no frameworks required)

## 🛠️ Built With

* **HTML5** – Structure of the website
* **CSS3** – Styling and responsive layout
* **JavaScript (ES6)** – Interactive functionality and cart management
